JavaScript — язык программирования

JavaScript — мультипарадигменный язык программирования, который чаще всего применяется как встраиваемый язык сценариев для веб-страниц. Он был создан Бренданом Эйхом в компании Netscape в 1995 году; по легенде первый прототип был написан примерно за десять дней. Изначально язык назывался Mocha, затем LiveScript, и лишь позднее получил маркетинговое имя JavaScript — несмотря на то, что с языком Java его связывает только часть синтаксиса, а не происхождение или семантика.

Чтобы избежать расхождений между браузерами, язык был стандартизирован организацией ECMA International. Стандарт называется ECMAScript (спецификация ECMA-262), и именно он описывает ядро языка, тогда как «JavaScript» — это наиболее распространённая реализация этого стандарта. Крупное обновление ECMAScript 2015 (известное также как ES6) добавило в язык классы, модули, стрелочные функции, ключевые слова let и const, а также промисы.

JavaScript имеет динамическую и слабую систему типов: тип значения определяется во время выполнения, а язык активно выполняет неявные приведения типов, что иногда приводит к неожиданным результатам. Управление памятью автоматическое, с использованием сборщика мусора. Объектная модель языка основана не на классах, а на прототипах: каждый объект может ссылаться на объект-прототип, от которого он наследует свойства и методы. Синтаксис классов, появившийся в ES6, является лишь удобной надстройкой над прототипным механизмом.

Важная особенность модели исполнения — однопоточность с циклом событий (event loop). Язык не блокирует выполнение в ожидании завершения операций ввода-вывода, а использует асинхронную модель на основе колбэков, промисов и синтаксиса async/await. Функции в JavaScript являются объектами первого класса и поддерживают замыкания, что делает возможным функциональный стиль программирования.

Долгое время JavaScript исполнялся только внутри браузера, где он управляет деревом документа (DOM) и обеспечивает интерактивность веб-страниц. Ситуация изменилась с появлением среды выполнения Node.js, построенной на движке V8 от Google: она позволила запускать JavaScript на сервере и вне браузера вообще. Современные движки (V8, SpiderMonkey, JavaScriptCore) используют JIT-компиляцию, превращая горячие участки кода в машинный код прямо во время исполнения, что заметно повысило производительность языка.

Экосистема JavaScript огромна: пакеты распространяются через менеджер npm, а для разработки пользовательских интерфейсов широко применяются библиотеки и фреймворки React, Vue и Angular. Файлы с кодом имеют расширение .js. Благодаря повсеместной поддержке в браузерах JavaScript остаётся фактически безальтернативным языком для фронтенд-разработки, а вместе с Node.js он используется и на бэкенде, что позволяет писать весь стек приложения на одном языке.

Пример вывода сообщения в консоль:

console.log("Hello, world!");

Несмотря на критику за исторические недостатки системы типов и неявные приведения, JavaScript благодаря стандартизации и постоянному развитию ECMAScript остаётся одним из самых востребованных языков в мире.

Долгое время совместимость между браузерами была болезненной проблемой: один и тот же код мог вести себя по-разному в разных браузерах, и разработчикам приходилось писать обходные пути. Эту боль во многом сняли библиотеки-посредники (например, jQuery в своё время) и последующая стандартизация. Сегодня обновления стандарта ECMAScript выходят ежегодно, и поддержка новых возможностей в браузерах появляется довольно быстро. Для использования самых свежих синтаксических конструкций в старых средах применяют транспиляторы вроде Babel, которые переводят современный код в более старую версию языка.

JavaScript часто критикуют за исторические причуды: слабая динамическая типизация и неявные приведения типов приводят к неожиданным результатам сравнения, а ряд решений был принят в спешке при создании языка. Частичным ответом на проблему типизации стал язык TypeScript — надмножество JavaScript со статическими типами. Несмотря на критику, JavaScript остаётся единственным языком, который изначально исполняется во всех веб-браузерах, и в связке со средой Node.js покрывает как клиентскую, так и серверную часть веб-приложений, что обеспечивает ему статус одного из самых распространённых языков в мире.
