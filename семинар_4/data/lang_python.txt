Python — язык программирования

Python — высокоуровневый язык программирования общего назначения, ориентированный на читаемость кода и повышение продуктивности разработчика. Создателем языка является Гвидо ван Россум. Работу над языком он начал в декабре 1989 года в Центре математики и информатики (CWI) в Нидерландах, а первый публичный релиз состоялся в 1991 году. Долгое время ван Россум исполнял роль «великодушного пожизненного диктатора» проекта и принимал ключевые решения о развитии языка вплоть до 2018 года. Python задумывался как потомок учебного языка ABC, способный к обработке исключений и взаимодействию с операционной системой.

Одной из самых узнаваемых особенностей языка является использование отступов для выделения блоков кода вместо фигурных скобок или ключевых слов begin/end. Эта синтаксическая особенность делает код визуально единообразным и заставляет придерживаться единого стиля оформления. Сообщество придерживается набора принципов под названием «The Zen of Python» (Дзен Питона); его текст выводится интерпретатором по команде import this. Среди этих принципов — «простое лучше, чем сложное» и «читаемость имеет значение».

Система типов Python динамическая и строгая: типы определяются во время выполнения, но неявные приведения между несовместимыми типами не происходят автоматически. Управление памятью полностью автоматическое и опирается на подсчёт ссылок, дополненный сборщиком мусора, который умеет разрывать циклические ссылки. Программист не управляет освобождением памяти вручную. Язык поддерживает несколько парадигм программирования: объектно-ориентированную, структурную (процедурную), а также элементы функционального программирования — функции являются объектами первого класса, поддерживаются лямбда-выражения, генераторы и списковые включения.

Эталонной реализацией языка является CPython — интерпретатор, написанный на C. Поскольку официального стандарта ANSI или ISO у языка нет, именно поведение CPython фактически играет роль спецификации. Существуют и альтернативные реализации, например PyPy с JIT-компиляцией. Развитие языка идёт через документы PEP (Python Enhancement Proposal); важнейшим из них считается PEP 8 — руководство по стилю оформления кода.

Большой переломной точкой в истории языка стал выход Python 3 в 2008 году. Третья версия была намеренно несовместима с веткой 2.x: изменилось поведение оператора деления, print стал функцией, строки по умолчанию стали хранить Unicode. Поддержка Python 2 была официально прекращена в 2020 году, и сегодня под словом «Python» практически всегда подразумевается ветка 3.x.

Сильной стороной Python является огромная стандартная библиотека и экосистема сторонних пакетов, устанавливаемых через менеджер pip из репозитория PyPI. Файлы с исходным кодом имеют расширение .py. Язык широко применяется в веб-разработке (фреймворки Django и Flask), в анализе данных и машинном обучении (библиотеки NumPy, pandas, PyTorch, scikit-learn), в автоматизации, скриптинге и системном администрировании, а также как первый язык в обучении программированию благодаря понятному синтаксису. Из-за интерпретируемой природы и накладных расходов на динамическую типизацию Python уступает в скорости компилируемым языкам, однако критичные по производительности участки часто выносят в расширения на C или вызывают через привязки к нативным библиотекам.

Пример программы «Hello, world!» на Python предельно короток и не требует объявления классов или функции main:

print("Hello, world!")

Простота и лаконичность подобных примеров — одна из причин, по которой язык остаётся одним из самых популярных в мире на протяжении многих лет.

История языка отмечена несколькими крупными вехами. Версия Python 2.0 вышла в 2000 году и принесла списковые включения и сборщик мусора, способный собирать циклические ссылки. Ветка 2.7, ставшая последней в линейке 2.x, поддерживалась особенно долго. Параллельно развивалась несовместимая ветка 3.x, и переход индустрии с Python 2 на Python 3 растянулся более чем на десятилетие. Начиная с версии 3.5 в язык добавили аннотации типов (type hints, PEP 484): они необязательны и не проверяются во время выполнения, но позволяют использовать внешние статические анализаторы вроде mypy.

У языка есть и заметные недостатки. Из-за глобальной блокировки интерпретатора (Global Interpreter Lock, GIL) в эталонной реализации CPython в каждый момент времени байт-код исполняет только один поток, что ограничивает использование многоядерных процессоров для вычислительных задач; обходными путями служат многопроцессность или вынос вычислений в нативные библиотеки. Тем не менее благодаря низкому порогу входа, читаемому синтаксису и колоссальной экосистеме Python многие годы удерживает лидерство в рейтингах популярности языков программирования и считается стандартом де-факто в науке о данных, машинном обучении и научных вычислениях.
