Rust — язык программирования

Rust — мультипарадигменный компилируемый язык программирования общего назначения, сочетающий парадигмы функционального и процедурного программирования с объектной системой, основанной на типажах. Работу над языком начал Грэйдон Хор в 2006 году как личный проект; в 2009 году к разработке подключилась компания Mozilla, и в 2010 году язык был официально представлен на конференции Mozilla Summit. Первая стабильная версия 1.0 вышла 15 мая 2015 года, после чего новые версии стали выходить каждые шесть недель.

Главные цели языка — безопасность, скорость и параллелизм. По производительности Rust сопоставим с C и C++, однако даёт значительно большую безопасность при работе с памятью. Достигается это без сборщика мусора и без затрат во время исполнения, благодаря принципу «абстракций с нулевой стоимостью» (zero-cost abstractions): высокоуровневые конструкции языка компилируются в столь же эффективный код, как если бы они были написаны вручную на низком уровне.

Центральная идея Rust — модель владения (ownership). У каждого значения есть ровно один владелец, и когда владелец выходит из области видимости, память автоматически освобождается. Значение можно временно «одолжить» через заимствования (borrow): либо много неизменяемых ссылок одновременно, либо ровно одну изменяемую — но не то и другое сразу. Корректность всех этих правил проверяется во время компиляции специальной подсистемой компилятора, которая называется borrow checker (проверщик заимствований). Именно borrow checker гарантирует отсутствие висячих указателей, гонок данных и обращений к освобождённой памяти ещё до запуска программы. По умолчанию все связывания неизменяемы; чтобы сделать переменную изменяемой, используется ключевое слово mut.

Объектная система Rust построена не на классах, а на типажах (traits). Типаж описывает набор методов, которые тип обязуется реализовать; реализация обозначается ключевым словом impl. Это похоже на интерфейсы в других языках, но мощнее: типажи могут содержать реализации по умолчанию и активно используются для перегрузки операторов и обобщённого программирования. Rust поддерживает обобщённые типы (generics), которые компилятор раскрывает через мономорфизацию, что делает их такими же быстрыми, как шаблоны C++.

На язык повлияли многие предшественники: функциональные возможности и система типов пришли из Haskell, OCaml и Standard ML, модель сообщений между потоками напоминает Erlang, а синтаксис блоков кода с фигурными скобками унаследован от C и C++. В свою очередь Rust сам повлиял на дизайн языка Swift. Компилятор Rust носит имя rustc и использует LLVM в качестве бэкенда для генерации машинного кода. Файлы с исходным кодом имеют расширение .rs, а стандартный менеджер пакетов и система сборки называются Cargo.

В декабре 2022 года Rust стал первым языком, помимо C и ассемблера, официально допущенным к разработке ядра операционной системы Linux. Язык широко применяется в системном программировании: при создании операционных систем, браузерных движков (например, Servo), сетевых сервисов, утилит командной строки и встраиваемого программного обеспечения. Несколько лет подряд Rust возглавлял опросы как «самый любимый язык» среди разработчиков.

Пример программы «Hello, world!» на Rust:

fn main() {
    println!("Hello, world!");
}

Таким образом, Rust предлагает редкое сочетание: скорость и контроль над памятью на уровне C++ вместе с безопасностью, которую обычно дают только языки со сборщиком мусора.

Освоение Rust традиционно считается непростым: модель владения и заимствований заставляет программиста с самого начала думать о времени жизни данных, и компилятор отвергает код, который в других языках скомпилировался бы, но потенциально приводил бы к ошибкам памяти. Этот «бой с borrow checker» — типичный опыт новичка. Однако взамен язык даёт сильную гарантию: если программа на безопасном Rust скомпилировалась, в ней по построению нет гонок данных и обращений к освобождённой памяти. Для случаев, когда нужен прямой контроль, существуют блоки unsafe, где часть проверок снимается и ответственность ложится на разработчика.

Экосистема языка организована вокруг менеджера пакетов Cargo и центрального репозитория crates.io, откуда подключаются библиотеки, называемые «крейтами» (crates). Cargo отвечает за сборку, тесты, зависимости и публикацию пакетов единообразным образом, что высоко ценится разработчиками. Помимо системного программирования, Rust всё активнее применяется в вебе через компиляцию в WebAssembly, в инструментах командной строки, в сетевых сервисах и в проектах, где безопасность памяти критична. Несколько лет подряд язык занимал первое место как «самый любимый язык» в крупных опросах разработчиков, что отражает редкое сочетание производительности и безопасности.
